Серверная часть проекта ачивок в реальной жизни.

Суть проекта: Люди любят играть в игры. В играх есть ачивки. Можно сделать жизнь веселее, внеся в неё ачивки. Делиться ими с друзьями, выполнять и радоваться.

Запуск:
1. Установить bottle и pymongo (см. requirements.txt)
2. Установить (это быстро) MongoDB: http://www.mongodb.org/downloads
3. >python routes.py
4. Открыть в браузере http://localhost:8082

На данный момент работает:
1. Регистрация и логин/логаут
2. Добавление друзей
3. Создание ачивок
4. Поиск ачивок
5. Удаление ачивок из своего списка.
Пока не реализовано из списка-минимум:
1. "Выполнение" ачивки
2. Просмотр профилей друзей
3. Добавление challenge'й друзьям

Техническое инфо:
В качестве БД используется MongoDB. Схема БД лежит в каталоге doc. Для соединения с БЖ используется драйвер pymongo
В качестве сервера обработки запросов используется фреймфорк bottle. Прописываются callbackи на различные запросы и выдаются страницы на основе шаблонов из директории views.
Внезапно, все прелести ООП не используются, т.к. по большей части вышло так, что python-код посредник между фронтендом и БД. В итоге остался процедурный стиль.
