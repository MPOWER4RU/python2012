Программа предназначена для заполнения сетки кроссворда словами из данного словаря.

На вход:
    - имя файла со спискм слов
    - имя файла с описание сетки кроссворда

пример: Main.py grid.txt dict.txt

формат сетки: * - клетка для символа
	      _ - пустая клетка(граница)

Результат вводится на консоль:
   - таблица с заполненным кроссвордом или сообщение об ошибке
