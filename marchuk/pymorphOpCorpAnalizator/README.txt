About:
Реализация морфологического анализатора русских слов на основе словарей Открытого корпуса русского языка (OpCorp)
[http://opencorpora.org] и пакета pymorphy. Также реализованы методы, позволяющие на основе тестов со снятой омонимией с
сайта OpCorp тестировать качество морфологического анализатора.

Requirements:
0) Python 2.6.6 (тестировалось только для него)
1) пакеты lxml и pymorphy (версии указаны в requirements.txt)
3) морфологический словарь в формате XML [http://opencorpora.org/dict.php]
4) тесты со снятой омонимией в формате XML [http://opencorpora.org/?page=downloads]
5) словари pymorphy [https://bitbucket.org/kmike/pymorphy/downloads/ru.sqlite-json.zip]

Usage:

Для использования морфологического анализатора - импортировать класс POCMorph из PymorphyWrapper. Для создания экземпляра
анализатора указать абсолютный путь до распакованного словаря pymorphy и до словаря для OpCorp. После этого можно
использовать методы get_graminfo и другие (см. документацию методов).

Для сравнения точности работы различных методов используются тексты со снятой омонимией в формате XML. Метод
getSentencesFromXML возвращает указанное количество предложений из указанного файла с текстами. Метод perfomanceTest
тестирует три указанных реализованных метода на текстах с сайта Открытого корпуса (работает долго).

Смотри также примеры и комментарии для реализованных методов.
