Для генерации точек множества используется программа GeneratePoints.py с параметрами:
  - количество точек
  - выходной файл
  - максимальные координаты (необязательный параметр, по умолчанию до 2^16)
  GeneratePoints.py 300 1.txt 1000
  
Основная программа - MinDisk.py - строит минимальную описывающую окружность и выводит её на экран
В качестве единственного параметра принимает имя входного файла с множеством точек
(для корректной работы алгоритма точки должны быть уникальными, то есть это должно быть именно множество)
Для получения в среднем линейной сложности алгоритма MinDisk используется процедура __permute(), перемешивающая точки множества за линейное время
